GUTENBERG HASH PROJECT

Takes in the Gutenberg database of books and stores every
line of every book into a hash table, such that when a client
inputs a word, the program finds each instance of the word and
returns the sentence it is found, the line number, and the book title.

BIOLOGIST'S GREP

Takes in a database of DNA sections and stores the information
in a Trie structure and finds whether client's DNA section
is already in the database and if not, gives the closest match and
the percentage match.
